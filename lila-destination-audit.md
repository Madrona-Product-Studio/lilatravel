**Lila Pick badge system:** All six guides use a `featured` prop on `ListItem` / `StayItem` components which renders a "Lila Pick" badge. For accommodations and restaurants, `featured` maps to `lilaPick: true` in JSON data files. For inline items (trails, wellness, culture, etc.), `featured` is set directly in JSX. Night Sky and Give Back sections generally do not use the badge system — items marked `--` below indicate no badge system applies to that section.
